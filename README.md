# 삼합전

바둑의 따내기, 오목의 5목, 알까기의 밀어내기를 한 판에 섞은 HTML + Canvas 하이브리드 전략 게임입니다.

GitHub Pages 배포용 정적 프로젝트라서 브라우저만 있으면 바로 플레이할 수 있습니다.

## 게임 특징

- `놓기`: 바둑처럼 활로를 계산해 상대 돌을 따낼 수 있습니다.
- `오목`: 5목을 완성하면 즉시 승리하면서 점수 보너스를 얻습니다.
- `알까기`: 난전 페이즈 이후 게이지가 가득 차면 자유 각도로 발사할 수 있습니다.
- `점수전`: 시작 시 선택한 목표 점수에 먼저 도달하면 승리합니다.
- `특수칸`: 중앙 왕점은 점수 보너스, 화점은 알까기 게이지 보너스를 줍니다.

## 플레이 방법

- 초반 `오프닝 페이즈`에서는 `놓기`만 가능합니다.
- 일정 턴이 지나면 `난전 페이즈`가 열리고 `알까기`를 사용할 수 있습니다.
- 알까기는 게이지가 `100`일 때만 사용할 수 있으며 사용 후 소모됩니다.
- 내 돌도 장외로 떨어질 수 있고, 그 경우 상대에게 점수가 들어갑니다.
- 따내기, 장외 탈락, 5목을 한 수에 겹치면 콤보 보너스를 얻습니다.

## 실행 방법

프로젝트 루트에서 [index.html](./index.html)을 브라우저로 열면 바로 실행됩니다.

## 배포

이 프로젝트는 정적 파일만 사용하므로 GitHub Pages로 쉽게 배포할 수 있습니다.

1. 저장소의 `Settings`로 이동합니다.
2. `Pages` 메뉴를 엽니다.
3. `Build and deployment`에서 `Deploy from a branch`를 선택합니다.
4. 브랜치를 `main`, 폴더를 `/ (root)`로 설정합니다.
5. 저장 후 잠시 기다리면 배포가 완료됩니다.
