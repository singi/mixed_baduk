# 삼합전

바둑의 따내기, 오목의 5목, 알까기의 밀어내기를 한 판에 섞은 HTML + Canvas 전략 게임입니다.

## 플레이

- `놓기`: 바둑처럼 활로를 계산해 상대 돌을 따낼 수 있습니다.
- `알까기`: 난전 페이즈 이후 게이지가 가득 차면 사용할 수 있습니다.
- `승리`: 시작 시 선택한 목표 점수를 먼저 달성하거나, 5목을 완성하면 즉시 승리합니다.

## 실행

브라우저에서 `index.html`을 열면 바로 플레이할 수 있습니다.

## 배포

이 프로젝트는 정적 파일만 사용하므로 GitHub Pages로 쉽게 배포할 수 있습니다.

1. GitHub에 새 저장소를 만듭니다.
2. 이 폴더를 `main` 브랜치로 푸시합니다.
3. GitHub 저장소의 `Settings > Pages`에서 `Deploy from a branch`를 선택합니다.
4. 브랜치는 `main`, 폴더는 `/ (root)`를 선택합니다.

배포 주소 예시:

`https://YOUR_USERNAME.github.io/YOUR_REPOSITORY/`
