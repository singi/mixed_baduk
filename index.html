행동 선택

난전 대기

알까기 모드에서는 돌이 선에 묶이지 않고 미끄러지며, 멈춘 뒤 가장 가까운 교차점에 다시 붙습니다.

대전 설정

AI는 5목, 따내기, 위험 차단을 우선으로 두는 가벼운 전략을 사용합니다.

알까기 세팅

백 돌 0

턴 로그

아직 기록이 없습니다.

흑 알까기 게이지 0 / 100
백 알까기 게이지 0 / 100

승리 조건

  • 시작 시 선택한 승리 점수를 먼저 얻으면 승리합니다.
  • 5목 완성은 즉시 승리이면서 4점을 줍니다.

혼합 규칙

  • 놓기는 바둑처럼 활로가 없는 상대 돌을 따내며, 돌 하나당 1점입니다.
  • 오프닝 6수 동안은 놓기만 가능하고, 이후 알까기가 해금됩니다.
  • 알까기는 게이지가 100일 때만 사용할 수 있고, 사용 후 게이지를 소비합니다.
  • 흑은 6목, 쌍삼, 쌍사를 금수로 취급합니다.
  • 직전과 같은 판 모양으로 되돌아가는 수는 패로 금지합니다.
  • 알까기로 판 밖으로 떨어진 상대 돌도 하나당 1점입니다.
  • 알까기는 자유 각도로 굴러가며 충돌하고, 판에 남은 돌은 가장 가까운 교차점에 정렬됩니다.
  • 중앙 왕점은 +1점, 화점은 알까기 충전 보너스를 줍니다.
  • 따내기, 탈락, 5목을 한 수에 겹치면 콤보 보너스를 얻습니다.
